# phoenix-os

Phoenix OSは、予想を自動化するアプリではなく、フェニックス斉藤が型を作るための思考整理ツールです。

## 現在の方針

- 型作成の入力項目はできる限り選択式
- 自由入力はメモ欄中心
- 選択内容から勝ち馬に求められる特徴を自動表示
- 最終判断は人間が行う
- CSV読み込みは補助機能

## Windowsで起動

```powershell
cd C:\Users\loron\Documents\Codex\2026-07-02\windows-next-js-phoenix-os-windows\phoenix-os

npm run dev
```
